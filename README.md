# SGBB VCS Training 2025

Repo này dùng để thực hành Git: commit, branch, merge, resolve conflict, tag release.
Không dùng cho mục đích production.

## Nội dung bài tập
- Tạo nhiều branch
- Commit các loại thay đổi
- Merge và xử lý conflict
- Tag phiên bản + ghi changelog
- Xoá & khôi phục file
